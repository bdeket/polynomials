Polynomials

 (require math/polynomials) package: polynomials

This library provides basic tools for calculations with polynomials. Polynomials are expressions of the form: a0.x^0+a1.x^1+a2.x^2+...+an.x^n. With n the degree of the polynomial.

1 Types, Predicates and Accessors

For now the polynomial is defined in 4 varieties. Exact-Rational epoly?, Real: flpoly?, Exact-Complex ecpoly? and Float-Complex flcpoly?. In what follows usually only the general poly functions are described. For specific ones add e,fl,ec or flc to poly.

procedure

(poly-reverse-coefficient P i) → Number

  P : poly?
  i : Integer
Get coefficient at position i. If i is too large or to small, 0 will be returned. The reverse form will return the coefficient starting from the highest degree. In other words: (poly-reverse-coefficient P i) is equivalent with (poly-coefficient P (- (poly-degree P) i))
